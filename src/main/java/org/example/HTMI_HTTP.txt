 Web: Базовые знания веб-технологий

1. Что такое HTTP и чем он отличается от HTTPS?
HTTP (HyperText Transfer Protocol) — протокол для передачи данных в вебе без шифрования, работает на порту 80.
 HTTPS (HTTP Secure) добавляет шифрование TLS/SSL и аутентификацию, использует порт 443, защищая от перехвата.

2. Какие основные методы HTTP-запросов вы знаете? В чем их различия?
Основные: GET (получение данных, безопасный, идемпотентный), POST (отправка данных, создание, не идемпотентный),
 PUT (полное обновление), PATCH (частичное обновление), DELETE (удаление), HEAD (метаданные без тела), OPTIONS (опции ресурса).

3. Что такое статус-коды HTTP? Назовите основные группы и приведите примеры.
Статус-коды HTTP — числовые ответы сервера, указывающие результат обработки запроса.
Группа 2xx (успех): запрос успешно обработан. Примеры: 200 OK (успешный запрос), 201 Created (ресурс создан), 204 No Content (успех без тела ответа).
Группа 3xx (перенаправление): требуется дополнительный запрос. Примеры: 301 Moved Permanently (постоянный редирект), 302 Found (временный редирект),
304 Not Modified (ресурс не изменился).
Группа 4xx (ошибки клиента): проблема в запросе клиента. Примеры: 400 Bad Request (неверный запрос), 401 Unauthorized (требуется авторизация),
403 Forbidden (запрещено), 404 Not Found (не найдено).
Группа 5xx (ошибки сервера): проблема на сервере. Примеры: 500 Internal Server Error (внутренняя ошибка), 502 Bad Gateway (ошибка шлюза),
503 Service Unavailable (сервис недоступен), 504 Gateway Timeout (таймаут шлюза).
Самые популярные коды: 200 OK (успех, чаще всего), 301/302 (редиректы), 404 Not Found (страница не найдена), 500 (серверная ошибка).

4. Объясните разницу между front-end и back-end разработкой.
Front-end — клиентская часть (UI с HTML/CSS/JS, видимая пользователю). Back-end — серверная (логика, БД, API, скрытая).

5. Что такое REST API? Какие у него основные принципы?
REST API — стиль для API с URI-ресурсами и HTTP-методами. Принципы: stateless, единообразный интерфейс, клиент-сервер,
кэшируемость, layered system, HATEOAS.

 HTML

1. Что такое HTML? Какова его основная структура?
HTML (HyperText Markup Language) — язык разметки для структуры страниц. Структура: <!DOCTYPE html>, <html>, <head>, <body>.

2. Для чего используются теги <head> и <body>?
<head> — метаданные (title, meta, CSS/JS ссылки), невидимый. <body> — видимый контент (текст, изображения).
